# Law-of-inertia
A physics-inspired spring animation built with HTML, CSS, and JavaScript that simulates realistic spring motion based on the principles of inertia and Hooke's Law. The spring stretches, compresses, and oscillates naturally, gradually coming to rest due to damping, closely mimicking real-world mechanical behavior. 
A realistic spring simulation built with HTML, CSS, and JavaScript that demonstrates inertia and Hooke's Law. Click or pull the spring to stretch it, then watch it oscillate naturally with damping, momentum, and smooth physics-based motion before returning to its equilibrium position. This project recreates real-world spring behavior using a simple physics engine and interactive animations.
